## Overview

`Flower Randomizer` is a small `React` + `Vite` project for generating floral artwork from a seed.
It currently supports three primary SVG-first modes: `Bouquet`, `Abstract`, and `Relief`.
An additional `Sculpture (Experimental)` mode is available as a seeded WebGL study, but it does not participate in the export or melody pipeline yet.

`Flower Randomizer` 是一个基于 `React` 和 `Vite` 的花艺生成项目。当前正式主线是 `Bouquet`、`Abstract` 与 `Relief` 三个 SVG-first 模式；`Sculpture (Experimental)` 是基于 WebGL 的实验性模式，暂不接入导出和 Melody 流程。

## Features

- Generate floral artwork from deterministic seeds
- Switch between `Bouquet`, `Abstract`, and `Relief` composition modes
- Restore and share the current artwork through the URL
- Export `Bouquet`, `Abstract`, and `Relief` artwork as `SVG` or `PNG`
- Explore `Sculpture (Experimental)` as an optional three.js-based scene mode
- Lightweight frontend-only project with no backend dependency

中文摘要：

- 使用可复现 seed 生成花艺画面
- 正式支持 `Bouquet`、`Abstract`、`Relief`
- 支持通过 URL 分享和恢复当前作品
- `Bouquet`、`Abstract`、`Relief` 支持 `SVG` / `PNG` 导出
- `Sculpture (Experimental)` 为实验模式，不接入导出和 Melody

## Local Development

### Requirements

- `Node.js`
- `npm`

### Setup

```bash
npm install
npm run dev
```

On Windows, you can also run:

```bat
start.bat
```

## Build

```bash
npm run build
```

The production build is written to `dist/`.

## Regenerating Preview Images

If you change the generator and want to refresh the README showcase images, run:

```bash
node scripts/generate-previews.mjs
```

The preview images are written to `docs/`.
If `Relief` should appear in the README showcase, regenerate or replace the preview set after capturing dedicated `Relief` outputs.

## Project Structure

```text
src/
  App.jsx                Main app shell and interactions
  ArtworkCard.jsx        SVG artwork rendering, relief layering, and export markup
  flowerGenerator.js     Bouquet, Abstract, and Relief generation logic
  ThreeDArtworkCard.jsx  Experimental sculpture renderer
  threeDGenerator.js     Experimental sculpture scene generation
  exportArtwork.js       SVG/PNG export helpers
  shareState.js          Share-link serialization and parsing
scripts/
  generate-previews.mjs  Generates README preview images
docs/
  preview-*.svg          Generated showcase images
  social-preview.png     Repository social preview image
public/
  flower-randomizer-logo.svg  Brand logo and favicon source
```

## Deployment

This is a static frontend app and can be deployed on any static hosting platform such as `Vercel`, `Cloudflare Pages`, or `GitHub Pages`.
